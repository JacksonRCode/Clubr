
  # GUI Prototypes and Navigation

  This is a code bundle for GUI Prototypes and Navigation. The original project is available at https://www.figma.com/design/jKgVm1qwJwK5mrttKprYYa/GUI-Prototypes-and-Navigation.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  

  ## Run from the project root (Clubr/) backend
    uvicorn api.index:app --reload --port 8000
